# Reaction-Diffusion System Simulation (反応拡散系の数値シミュレーション)

活性因子と抑制因子の相互作用によるパターン形成（チューリングパターン）をシミュレーションし、可視化したプロジェクトです。学部4年時の研究の一環として作成しました。

## 💡 概要 (Overview)
本プログラムは、自然界のパターン形成を記述する**反応拡散系（例：Gray-Scottモデル）**の偏微分方程式を、C言語を用いて数値計算したものです。

* **使用言語**: C (計算エンジン), gnuplot (可視化)
* **計算手法**: 中心差分法による離散化
* **シミュレーション条件**: $256 \times 256$ グリッド、周期境界条件
* **アップデート**: 学部時代の研究コードをベースに、大学院でgnuplotを用いた自動動画生成パイプライン（シェルスクリプト/gnuplot script）を追加実装し、可視化を強化しました。

## 🛠 工夫した点・アピールポイント
1. **数理モデルの忠実な実装と最適化**
   複雑な偏微分方程式を安定的に解くための時間刻み幅（dt）の設定や、多重ループ内でのメモリメモリアクセスを意識した実装を行い、計算速度と精度の両立を図りました。
2. **自動化パイプラインの構築**
   数値データ（.txt）から大量の連番画像を生成し、動画化するまでの工程を `png3d.gnu` 等で自動化。研究効率を高めるツール制作能力をアピールします。
3. **ゲーム・CG制作への応用（テクニカルアーティスト視点）**
   本技術は、シェーダー（HLSL/GLSL）を用いたリアルタイムの模様生成や、プロシージャルなテクスチャ作成のアルゴリズムとして直接応用可能です。数学的背景に基づいた表現制作（Math-based Art）を得意としています。
