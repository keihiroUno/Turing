# Reaction-Diffusion System Simulation (反応拡散系の数値シミュレーション)

活性因子と抑制因子の相互作用によるパターン形成（チューリングパターン等）をシミュレーションし、可視化したプロジェクトです。学部4年時の研究の一環として作成しました。

## 💡 概要 (Overview)
本プログラムは、自然界に見られる動物の模様などを数学的にモデル化した「反応拡散系」の方程式を、C言語を用いて数値計算したものです。計算結果を `png3d.gnu` ファイルを用いて自動的に画像化・動画化するパイプラインまで構築しています。

* **使用言語・ツール**: C, gnuplot
* **計算手法**: 中心差分法による偏微分方程式の離散化

## 🛠 工夫した点・アピールポイント
1. **アルゴリズムの実装力**
   複雑な数式（偏微分方程式）をコードに落とし込み、計算速度やメモリ管理を意識したC言語プログラミングを行いました。
2. **データの可視化（自動化）**
   計算結果の単なる数値出力に留まらず、gnuplotのスクリプトを記述することで、大量の計算結果から自動で視覚的なアニメーションを生成するフローを構築しました。
3. **ゲーム業界・CGへの応用視点**
   この「数式から自然な模様をプロシージャル（自動）生成する」技術は、ゲームにおけるテクスチャの自動生成や、自然現象のVFXシミュレーション（テクニカルアーティスト領域）の基礎となる考え方です。
